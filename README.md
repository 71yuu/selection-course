# selection-course

## 概述

这是一个简单的教务管理系统，使用了 Spring、Spring MVC 、MyBatis 技术。该系统有三个角色：管理员、教师、学生，分别有不同的功能，管理员可以管理课程、教师、学生、院系、选课。教师可以查询课程信息、修改密码。学生可以查询选课信息、修改密码。一个很粗略的项目，很多功能都没有完善。



## 开发环境

- 开发工具：Intellij IDEA
- 数据库：MySQL 5.7
- Java SDK：JDK 1.8.192



## 项目管理工具

- 项目构建：Maven

## 后台主要技术栈

- 基础框架：Spring
- Web 框架：Spring MVC
- ORM 框架：MyBatis
- 数据库连接池：Alibaba Druid



## 前台

- AdminLTE 模板
